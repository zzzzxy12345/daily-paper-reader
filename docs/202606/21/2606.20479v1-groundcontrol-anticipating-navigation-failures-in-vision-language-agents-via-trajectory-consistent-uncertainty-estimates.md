## 摘要
视觉语言导航代理在基准任务上实现了具有竞争力的平均成功率，然而失败往往源于可预测的轨迹级崩溃，例如震荡、停滞或低效绕行。因此，可靠部署需要的不确定性信号应能在执行过程中预判新兴的失败动态，而非仅反映瞬时动作熵。我们引入了GroundControl，这是一种轨迹一致的不确定性估计器，定义为从名义上以目标为导向的距目标距离动力学中统计偏差的聚合，贯穿整个情节。GroundControl使用恒定速度卡尔曼滤波器对距离演化进行建模，并将归一化的创新统计量与补充轨迹特征（捕捉进度、单调性、路径效率和震荡行为）相结合。由此产生的不确定性分数反映的是导航行为中的几何和时间不一致性，而非局部预测分散性。为了独立于任务成功评估不确定性质量，我们正式定义了选择性风险覆盖导航（SRCN）协议，该协议通过风险覆盖曲线以及AURC/E-AURC总结来衡量不确定性分数按失败或低效程度对情节排序的有效性。在五个EB-Navigation分割（N=300个情节）中，基于成功的选择性风险下，轨迹一致性不确定性实现了接近预言的排序，GPT-4o模型的加权平均E-AURC_SR=0.0024，大幅优于基于熵、共形和启发式的基线。在基于SPL的选择性评估下，GroundControl在各个模型和导航分割中始终获得最低的AURC和E-AURC。这些结果表明，对目标导向动力学偏差进行建模，为预测视觉语言代理中的导航失败提供了可解释且稳健的信号。

## Abstract
Vision-language navigation agents achieve competitive average success on benchmark tasks, yet failures often arise through predictable trajectory-level breakdowns such as oscillation, stagnation, or inefficient detours. Reliable deployment, therefore, requires uncertainty signals that anticipate emerging failure dynamics during execution rather than reflect only instantaneous action entropy. We introduce \emph{GroundControl}, a trajectory-consistent uncertainty estimator defined as statistical deviation from nominal goal-directed distance-to-goal dynamics aggregated over an episode. GroundControl models distance evolution using a constant-velocity Kalman filter and combines normalized innovation statistics with complementary trajectory features capturing progress, monotonicity, path efficiency, and oscillatory behavior. The resulting uncertainty score reflects geometric and temporal inconsistency in navigation behavior rather than local prediction dispersion. To evaluate uncertainty quality independently of task success, we formalize \emph{Selective Risk--Coverage Navigation (SRCN)}, a protocol that measures how effectively an uncertainty score ranks episodes by failure or inefficiency using risk--coverage curves and AURC / E-AURC summaries. Across five EB-Navigation splits ($N=300$ episodes), trajectory-consistent uncertainty achieves near-oracle ordering under success-based selective risk, with weighted-average $\mathrm{E\text{-}AURC}_{\mathrm{SR}}=0.0024$ for the GPT-4o model, substantially outperforming entropy-, conformal-, and heuristic baselines. Under SPL-based selective evaluation, GroundControl consistently achieves the lowest AURC and E-AURC across models and navigation splits. These results show that modeling deviation from goal-directed dynamics provides an interpretable and robust signal for anticipating navigation failures in vision-language agents.

## 2. 论文提出的方法论

### 核心思想
- **轨迹一致性不确定性**：成功的导航应表现出平滑、近似单调递减的距离‑目标信号。轨迹级别的偏差（如停滞、振荡、发散）可通过统计假设检验检测出来。GroundControl 通过恒定速度卡尔曼滤波器（Constant‑Velocity Kalman Filter）对距离演化建模，并以归一化创新统计量、后验协方差增长等指标量化偏离名义目标导向运动的程度，再融合多个轨迹特征得到最终的情节级不确定性分数。

### 关键技术细节
1. **状态定义**：状态向量 \(x_t = [d_t, \dot{d}_t]^T\)，其中 \(d_t\) 是时刻 \(t\) 到目标的欧氏距离，\(\dot{d}_t\) 是瞬时变化率。
2. **线性高斯动力学**：
   - 转移矩阵 \(F = \begin{bmatrix}1 & 1\\0 & 1\end{bmatrix}\)，观测矩阵 \(H = [1, 0]\)。
   - 过程噪声 \(w_t \sim \mathcal{N}(0, Q)\)，观测噪声 \(v_t \sim \mathcal{N}(0, \sigma_r^2)\)。
   - 初始状态：\(\hat{x}_0 = [d_0, 0]^T\)，\(\Sigma_0 = \text{diag}(0.01, 1.0)\)。
3. **卡尔曼滤波步骤**：预测 → 计算创新（innovation）\(\nu_t = d_t - H\hat{x}_{t|t-1}\) → 更新。归一化创新平方 \(\nu_t^2 / S_t\)（服从卡方分布）作为统计一致性检验。
4. **每步不确定性分数**：
   \[
   H_t = \lambda_1 \log \det(\Sigma_{t|t} + \epsilon I) + \lambda_2 H(\alpha_t) + \lambda_3 \frac{\nu_t^2}{S_t}
   \]
   - 第一项：后验协方差的对数行列式，度量滤波不确定性。
   - 第二项：执行动作的熵 \(H(\alpha_t)\)，\(\alpha_t\) 为所选动作的 softmax 概率，捕捉动作级犹豫。
   - 第三项：归一化创新，检测动态偏差。
   - 固定权重 \((1.0, 0.5, 0.8)\)。
5. **情节级得分**：对每步 \(H_t\) 取最大值，再通过 sigmoid 映射到 [0,1]：\(\tilde{s}_{kf}^i = \sigma(\max_t H_t)\)。
6. **轨迹特征融合**（各归一化到 [0,1]，值越大表示偏差越大）：
   - \(f_{\text{prog}}\)：净进度不足（归一化距离减少量）。
   - \(f_{\text{mono}}\)：非单调性（距离增大的步数比例）。
   - \(f_{\text{eff}}\)：路径效率（位移/路径长度）。
   - \(f_{\text{osc}}\)：震荡行为（动作反转对的比例）。
7. **最终不确定性分数**：
   \[
   U_i = w_{skf} \tilde{s}_{kf}^i + w_{\text{prog}} f_{\text{prog}}^i + w_{\text{mono}} f_{\text{mono}}^i + w_{\text{eff}} f_{\text{eff}}^i + w_{\text{osc}} f_{\text{osc}}^i
   \]
   权重 \((0.30, 0.25, 0.20, 0.15, 0.10)\) 固定，不通过学习获得，以保持可解释性。

8. **SRCN 协议**：选择性风险覆盖导航（Selective Risk–Coverage Navigation）——给定不确定性排序，逐步排除高不确定性情节，计算保留集上的失败率（或 1‑SPL），绘制风险‑覆盖曲线，用 AUC（AURC）和 Excess‑AURC（E‑AURC）量化排序质量（E‑AURC=0 表示完美排序）。
