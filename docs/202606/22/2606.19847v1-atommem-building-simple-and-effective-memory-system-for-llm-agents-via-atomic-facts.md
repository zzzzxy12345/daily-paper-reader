## 摘要
大型语言模型（LLM）展现出强大的推理和生成能力，但其固定上下文窗口限制了跨多会话交互的长期信息积累与重用。现有记忆增强系统通常以粗糙且不稳定的方式构建记忆，依赖于低效的记忆表示或不可控的无约束更新。为应对这些挑战，我们提出AtomMem，一种面向高价值密度存储与稳定记忆演化的长期记忆系统。AtomMem引入事实执行器（Fact Executor），从长程交互中选择性提取高价值的原子事实，作为高效记忆表示。随后，AtomMem将这些事实组织为分层事件结构与时间剖面，捕获连贯的情景语境并追踪随时间动态演化的用户属性。在检索过程中，系统激活关联记忆图以连接碎片化记忆。在LoCoMo基准上的实验证实，AtomMem在各种推理任务中取得了最先进的性能，为部署智能个性化代理提供了可扩展且经济可行的解决方案。

## Abstract
Large language models (LLMs) demonstrate strong reasoning and generation abilities, but their fixed context windows limit long-term information accumulation and reuse across multi-session interactions. Existing memory-augmented systems often construct memory in a coarse and unstable manner, relying on inefficient memory representations or unstable unconstrained updates. To address these challenges, we propose AtomMem, a long-term memory system designed for value-dense storage and stable memory evolution. AtomMem introduces a Fact Executor, which selectively extracts high value atomic facts from long form interactions to serve as highly efficient memory representations. Subsequently, AtomMem organizes these facts into hierarchical event structures and temporal profiles, capturing coherent episodic contexts and tracking dynamically evolving user attributes over time. During retrieval, the system activates an associative memory graph to connect fragmented memories. Experiments on the LoCoMo benchmark confirm that AtomMem achieves state-of-the-art performance across various reasoning tasks, offering a scalable and economically viable solution for deploying intelligent personalized agents.

## 2. 论文提出的方法论

- **核心思想**：提出 AtomMem，一个以“原子事实”为核心的长期记忆系统。它将原始对话流转化为结构化的、自包含的原子事实，并组织成分层事件记忆和时间画像，在检索时通过关联记忆图激活相关记忆，实现稳定、高效、可扩展的记忆管理。

- **关键技术细节**：
  - **原子事实提取**：使用经过监督微调（SFT）的 Qwen3-14B 模型作为事实执行器，对原始对话进行去噪、指代消解和时间锚定，输出自包含的原子事实。每个事实包含文本内容、嵌入向量、参与者、关键词、时间信息以及关联事件 ID 等结构化元数据。
  - **事实验证与更新**：新事实生成后，通过混合相似度（语义嵌入 + 关键词 Jaccard）检索最相关候选事实，LLM 判断是否存在重复或冲突，存储唯一新颖内容并更新已有事实。
  - **事件记忆构建**：将相关原子事实聚合为事件块，每个事件包含摘要、成员事实 ID、参与者、关键词和时间跨度。通过 LLM 判断新事实应吸收到现有事件还是创建新事件。
  - **时间画像建模**：基于会话级批量处理，从事实中识别隐含长期特征，构建用户画像条目，支持历史版本记录以追踪偏好演化。
  - **关联回忆**：构建以原子事实为节点的记忆图，包含三种边：
    - 实体边（关键词重叠，带 IDF 加权）
    - 事件边（同一事件成员，带大小惩罚）
    - 时序边（相邻对话轮次，位置距离衰减）
  - 检索时，先通过分层混合检索得到种子事实，再在子图上执行带重启的随机游走（Personalized PageRank）以传播激活分数，最终选出 top-k 事实用于生成响应。
  - **响应生成**：结合意图分析、分层检索（主检索+补偿检索+关联检索）以及画像增强，将检索到的记忆与查询一起输入 LLM 生成答案。

- **公式或算法流程（文字说明）**：
  - 混合相似度：`Sh(x,y) = α·sim_e(v_x,v_y) + β·Jac(K_x,K_y)`，平衡语义和关键词。
  - 更新函数：`(c'_new, U) ← LLM(c_new ∥ C_ret)`，生成残差内容与冲突更新。
  - 图边权重：实体边用 IDF 加权关键词重叠；事件边用成员数量惩罚；时序边用指数衰减。
  - 随机游走：`r^{(t+1)} = ηp + (1-η)P^T r^{(t)}`，其中 p 为种子重启分布，P 为多通道归一化转移矩阵。

## 7. 优点（方法或实验设计上的亮点）

- **记忆表示创新**：提出“原子事实”作为基本语义单元，通过 SFT 实现高质量去噪和上下文消歧，显著优于原始对话或粗糙摘要。
- **层次化记忆架构**：事件记忆+时间画像+关联图，兼顾细节保真度、情景完整性和长期演化追踪。
- **稳定更新机制**：通过事实验证（去重、冲突解决）和画像历史版本，避免无限制重写带来的噪声累积。

- **实验公平性**：统一所有基线使用相同骨干模型、嵌入模型和 top-k 设置，重新实现而非直接引用原论文结果。
- **开源贡献**：公开代码和训练数据集，促进可复现研究。
