## 摘要
现实世界中的决策常常需要同时优化多个相互竞争的目标。在强化学习（RL）中，这通常通过标量化函数将奖励信号组合成单个标量目标来解决，但这种方法可能很脆弱：权重的微小变化可能导致截然不同的策略。多目标强化学习（MORL）则生成显式表示目标之间权衡的策略集。然而，这些策略通常仅通过其价值向量呈现给决策者，这可能会掩盖显著的行为变化：仅通过预期回报评估时，产生不同轨迹的策略可能看起来难以区分。我们提出了一种探索性诊断工作流程，该工作流程自动突出显示帕累托前沿上目标值本身无法揭示的行为变化，提供定量和可视化工具以支持策略检查。我们在简单网格示例上验证了我们的方法，并将其扩展到连续控制基准测试，证明了该方法随着问题复杂性的增加仍然有效。

## Abstract
Real-world decision-making often requires optimizing multiple competing objectives simultaneously. In reinforcement learning (RL), this is typically addressed by combining reward signals into a single scalar objective via a scalarization function, which can be fragile: small changes in the weights can induce drastically different policies. Multi-objective reinforcement learning (MORL) instead produces sets of policies that explicitly represent trade-offs between objectives. However, these policies are typically presented to the decision maker only through their value vectors, which can obscure substantial behavioral variation: policies that induce distinct trajectories may appear indistinguishable when evaluated solely by expected returns. We propose an exploratory diagnostic workflow that automatically highlights behavioral variation along the Pareto front that objective values alone do not reveal, providing both quantitative and visual tools to support policy inspection. We validate our approach on simple grid examples and scale it to continuous control benchmarks, demonstrating that it remains effective as problem complexity increases.

### 论文核心问题与整体含义（研究动机和背景）

- **核心问题**：多目标强化学习（MORL）中，决策者通常仅依赖帕累托前沿（Pareto Front）上的价值向量来选择策略，但不同策略可能产生行为差异极大的轨迹，而目标空间（Objective Space）的近邻却无法反映这些差异。这种“目标-行为错位”（Objective-Behavior Misalignment）可能导致决策者选中数值上看似优秀但实际行为不理想甚至危险的策略。
- **研究背景**：实际应用（如机器人、自动驾驶）中，奖励函数通常通过试错标量化权重设计，权重微小变化可导致策略行为剧变（Vamplew et al., 2022; Booth et al., 2023）。现有MORL评估只关注目标空间的收敛性和多样性（如超体积），未系统分析行为差异。作者通过改进的“左右深海宝藏”（Left–Right DST）示例直观展示：目标相近的策略可能分别左行或右行，行为截然不同，但价值向量相同，因此必须引入行为空间的诊断。

- **核心思想**：构建一个模块化工作流，自动量化并可视化帕累托前沿上策略间行为差异与目标差异的不一致性，标记出“目标空间中邻近但行为空间遥远”的策略对，供决策者人工检查。

    2. **行为编码与聚合**：使用基于Transformer的对比学习编码器（Mone et al., 2026）将每条轨迹映射为低维嵌入（3维），再通过平均聚合得到每个策略的行为嵌入z(π)。编码器采用复合损失L_total = αL_CLS + βL_DIM + γL_SEG + δL_PAIR + L_VC（方差-协方差正则化），以学习行为相关且几何结构良好的嵌入空间。

### 论文的主要结论与发现

1. **目标-行为错位确实存在**：在Left–Right DST中，信任度/连续性仅约0.5，远低于Smooth DST的近1，说明目标空间无法捕获行为差异。
2. **Transformer编码器可有效提取行为特征**：手动嵌入与Transformer嵌入对比的信任度/连续性接近0.98，说明编码器学习到的行为关系与人类设计一致。

4. **行为差异可通过Lipschitz散点图自动标记**：左上区域点对应的高行为差异/低目标差异对提示决策者需人工审查。
5. **方法局限性**：工作流本身不能判断行为差异是否有害或目标是否不完整，需要结合外部利益相关者偏好信息。
