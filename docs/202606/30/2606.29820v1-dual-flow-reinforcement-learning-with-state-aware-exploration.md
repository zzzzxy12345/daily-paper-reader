## 摘要
在复杂的连续控制强化学习任务中，多模态最优动作常与不确定的多模态回报分布同时出现，这使得可靠的价值估计和多模态探索变得具有挑战性。现有使用单峰高斯分布的价值估计方法限制了表达能力并产生有偏估计。近期生成式策略可以表示多模态动作，但往往坍缩到少数模态，并对动作空间中高价值区域探索不足。受这些挑战启发，我们提出双流强化学习（Dual-Flow RL），一种统一的演员-评论家框架，使用条件流匹配（CFM）联合建模连续回报分布和多模态策略分布。该设计支持可靠的价值估计和持续的多模态探索。为进一步增强探索，我们引入熵-协方差探索调节器（ECER），利用策略熵和动作不确定性的协方差实现状态感知的探索调节。在DeepMind Control Suite和Humanoid-Bench上的实验表明，双流强化学习在大多数任务上达到了最先进的性能，显著优于先前的基于扩散和基于流的方法。

## Abstract
In complex continuous-control reinforcement learning tasks, multimodal optimal actions often coincide with uncertain, multimodal return distributions, making reliable value estimation and multimodal exploration challenging. Existing value estimation methods using unimodal Gaussians restrict expressiveness and yield biased estimates. Recent generative policies can represent multimodal actions but often collapse to a few modes and under-explore high-value areas of the action space. Motivated by these challenges, we propose Dual-Flow RL, a unified actor-critic framework that jointly models a continuous return distribution and a multimodal policy distribution using conditional flow matching (CFM). This design supports reliable value estimation and sustained multimodal exploration. To further enhance exploration, we introduce an Entropy-Covariance Exploration Regulator (ECER) that enables state-aware exploration regulation leveraging policy entropy and action-uncertainty covariance. Experiments on DeepMind Control Suite and Humanoid-Bench show that Dual-Flow RL achieves state-of-the-art performance on most tasks, significantly outperforming prior diffusion-based and flow-based methods.

- **整体意义**：现有工作分别改善价值估计或策略表示，但缺乏统一框架；探索机制也多是全局固定的，无法根据状态自适应调节。本文旨在同时解决这两个问题，实现可靠价值估计与可持续多模态探索的统一。

## 2. 方法论

### 2.1 核心思想
- 提出 **双流强化学习（Dual-Flow RL）**，一种统一的演员-评论家框架，采用条件流匹配（Conditional Flow Matching, CFM）**联合建模连续回报分布和连续多模态策略分布**。
- 引入 **熵-协方差探索调节器（ECER）**，利用策略熵和动作-不确定性协方差，实现状态感知的自适应探索调节。

### 2.2 关键技术细节

#### a) 基于流匹配的评论家（Distributional Critic）
- 将回报分布建模为状态-动作条件连续分布 \(Z(s,a)\)，通过ODE从基噪声 \(z_0 \sim \mathcal{N}(0,1)\) 演化得到回报样本 \(z_1\)：
  \[
  \frac{dz_t}{dt} = v_z(s, a, z_t, t),\quad z_1 = z_0 + \int_0^1 v_z\, dt.
  \]
- 使用线性插值路径和CFM训练速度场 \(v_z\)，最小化与TD目标分布之间的流匹配损失（式(15)）。

#### b) 基于流匹配的演员（Flow-based Policy）
- 策略同样通过CFM建模：从 \(a_0 \sim \mathcal{N}(0,I)\) 经ODE生成动作 \(a_\pi = a_1\)：
  \[
  \frac{da_t}{dt} = v_\pi(s, a_t, t).
  \]
- 策略优化目标（式(19)）包含两部分：
  - 最大化策略动作的期望回报 \(Q(s, a_\pi(s))\)；
  - 利用**优势加权对齐**：对重放缓冲区中优势更大的动作赋予更高权重（\(w(s,a)\)），引导策略分布向高质量行为靠近。

#### c) 熵-协方差探索调节器（ECER）
- 在策略输出后添加可学习的状态相依探索噪声：\(a_e = a_\pi + \sigma(s) \odot \epsilon,\ \epsilon \sim \mathcal{N}(0,I)\)。
- 自适应调节探索强度 \(\sigma(s)\)：
  - **熵门控**：基于GMM估计的策略熵 \(H(s)\)，当熵低于目标时增大探索（\(g_H\)）；
  - **协方差门控**：计算动作log密度与回报不确定性之间的相关性 \(\rho(s)\)，当相关性高时（表明探索集中在高不确定性区域），减小探索惩罚（\(g_D\)）。
  - 联合门控 \(g = g_H g_D\) 控制有效正则化系数 \(\lambda_{\text{eff}} = \lambda_0 / (g^2 + \epsilon)\)。
- ECER训练目标（式(28)）：最大化优势加权对数似然 + 探索尺度正则化。

### 2.3 算法流程
- 算法1（Dual-Flow RL）：
  1. 每一迭代：采样状态 → 策略生成动作 → ECER加噪 → 与环境交互 → 存入回放缓冲区。
  2. 每次更新：采样batch → 更新评论家流匹配损失（式(26)）→ 更新策略流匹配损失（式(27)）→ 更新ECER（式(28)）→ 软更新目标网络。每隔U步更新门控参数。

## 6. 主要结论与发现

- Dual-Flow RL在DMC和H-Bench上**大多数任务达到SOTA**，显著超越扩散和流基线。例如：
  - DMC Humanoid-run：比FlowRL高31.6%，比SAC高112.3%；
  - H-Bench H1-reach：比FlowRL高30.5%，比QVPO高56.8%。
- 分布评论家相比于期望值评论家可带来更快收敛和更高最终回报。
- ECER相比其他探索调节方法（DACER、固定噪声缩放）显著提升性能。
- 方法对流步数不敏感（浅层推理已足够），对正则系数在中等范围内鲁棒。

- **实验全面**：涵盖多个连续控制基准和大量基线，消融分析严谨。
