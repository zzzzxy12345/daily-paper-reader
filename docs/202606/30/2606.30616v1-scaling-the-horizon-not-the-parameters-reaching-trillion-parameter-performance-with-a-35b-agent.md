## 摘要
我们提出Agents-A1，一个35B混合专家智能体模型，通过扩展智能体地平线达到万亿参数级别的性能。我们从两个角度研究智能体地平线扩展：扩展长程轨迹和扩展异构智能体能力。为支持这一目标，我们构建了一个长程知识-行动基础设施，连接外部知识、行动、观察和验证器结果，生成长度平均为45K token的智能体轨迹。基于此，我们采用三阶段流程训练Agents-A1。首先，进行全领域监督微调，使基座模型对齐广泛的智能体行为。其次，训练领域级教师模型以捕捉每个领域的专业知识。第三，我们提出一种基于多教师领域路由的在线策略蒸馏，结合显著词汇对齐，以提高跨领域知识迁移效率，将六个异构领域统一到一个可部署的学生模型中。Agents-A1在长程智能体基准上取得了强大而广泛的表现。与万亿参数模型如Kimi-K2.6和DeepSeek-V4-pro相比，Agents-A1在SEAL-0 (56.4)、IFBench (80.6)、HiPhO (46.4)、FrontierScience-Olympiad (79.0)和MolBench-Bind (56.8)上取得领先结果，并在SciCode (44.3)、HLE (47.6)和BrowseComp (75.5)上保持高度竞争力。我们希望这项工作为社区提供一条实用路径，使用35B智能体扩展地平线，能够在长程任务上达到或匹配万亿参数模型的性能。

## Abstract
We introduce Agents-A1, a 35B Mixture-of-Experts Agentic Model that reaches trillion-parameter-level performance by scaling the agent horizon. We investigate agent-horizon scaling from two perspectives: scaling long-horizon trajectories and scaling heterogeneous agent abilities. To support this goal, we build a long-horizon knowledge-action infrastructure that connects external knowledge, actions, observations, and verifier outcomes, producing agentic trajectories with an average length of 45K tokens. Based on this, we train Agents-A1 with a three-stage recipe. First, we perform full-domain supervised fine-tuning to align the base model with broad agentic behaviors. Second, we train domain-level teacher models to capture specialized expertise in each domain. Third, we propose a multi-teacher domain-routed on-policy distillation with salient vocabulary alignment to improve knowledge transfer efficiency across different domains, unifying six heterogeneous domains into one deployable student model. Agents-A1 achieves strong and broad performance for long-horizon agent benchmarks. Compared with 1T-parameter model such as Kimi-K2.6 and DeepSeek-V4-pro, Agents-A1 achieves leading results on SEAL-0 (56.4), IFBench (80.6), HiPhO (46.4), FrontierScience-Olympiad (79.0), and MolBench-Bind (56.8), and remains highly competitive on SciCode (44.3), HLE (47.6) and BrowseComp (75.5). We hope this work provides the community with a practical path for scaling the horizon using a 35B agent that can reach or match the performance of 1T models on long-horizon tasks.

## 1. 核心问题与整体含义（研究动机与背景）

- **核心问题**：当前大语言模型（LLM）主要依赖**参数规模扩展**（如万亿参数模型）来提升推理、工具调用等智能体能力。这种路径成本极高（算力、数据、训练），且难以被中小团队复现。
- **替代思路**：本文提出**扩展智能体交互范围（agent horizon）** 而非参数——即让模型在更长的交互轨迹中规划、执行、验证、反思，通过**过程级监督**而非仅依赖参数内化知识来提升能力。
- **关键瓶颈**：
  - 缺乏连接外部知识、行动、观察和验证信号的统一基础设施；
  - 跨领域异构能力（搜索、工程、科学、指令遵循、工具使用）在单一模型中难以融合，存在推理模式冲突。

## 2. 方法论：核心思想、关键技术细节、算法流程

### 核心思想
- 不再追求参数量，而是让模型在**长程交互轨迹**中学习，通过**过程监督**（而非仅最终答案）积累能力。
- 构建**知识-动作图（Knowledge-Action Graph, KAG）** 保存轨迹状态，并采用**三阶段训练**融合多领域专家能力。

### 关键技术细节

#### (1) 长程知识-动作基础设施（KAG）
- 定义：给定领域 \(B_d\)，KAG 为四元组 \(G_d = (C_d, A_d, O_d, V_d)\)，分别代表领域语料、动作空间、观察空间、验证器集合。
- **过程级记录**：每个决策步骤表示为 \((s_t, a_t, o_t, v_t)\)，其中 \(s_t\) 为当前状态，\(a_t\) 为动作（如搜索、代码执行），\(o_t\) 为观察结果，\(v_t\) 为验证信号。失败轨迹也被保留，用于负样本学习。
- **自对弈扩展**：通过提议者-求解者-验证者游戏，不断生成、验证、扩充高质量轨迹，平均轨迹长 **45K tokens**。

#### (2) 三阶段训练流程
1. **全领域监督微调（SFT）**：基于 Qwen3.5-35B-A3B，在多领域长程轨迹（约10万条）上微调，学习基础智能体行为。
2. **领域教师模型训练**：针对六类场景（搜索、机器学习工程、科学推理、指令遵循、工具调用、通用Agent），分别用 SFT + 强化学习（GRPO）训练教师模型：
   - 搜索教师：SFT + GRPO，奖励包括正确性、效率惩罚、重复搜索惩罚、格式校准。
   - 科学教师：两阶段 SFT（先纯推理，后工具增强）。
   - 指令遵循教师：两阶段 RL（先细粒度指令，后长上下文学习）。
   - 工具调用教师：SFT + RL，采用非对称优势设计（PAPO风格）。
3. **多教师域路由在线蒸馏（OPD）**：
   - 学生模型保留通用学步能力，对每条样本按领域标签选择对应教师进行监督。
   - **显著词汇对齐（Salient Vocabulary Alignment, SVA）**：在学生生成轨迹上，将教师分布与学生分布仅在教师高概率的Top-K token集合上做反向KL散度，避免全词汇对齐带来的噪声。
   - **领域归一化损失**：按活跃域平均损失，再跨域平均，防止高频域主导更新。

## 6. 主要结论与发现

1. **35B模型可达到万亿参数级性能**：Agents-A1在SEAL-0、IFBench、HiPhO、FS-O、MolBench-Bind上超越 Kimi-K2.6 和 DeepSeek-V4-pro；在GAIA、XBench、BsrowseComp上持平或接近。
2. **扩展交互范围比扩展参数更有效**：通过构建长程知识-动作基础设施，模型能在平均45K token的轨迹中学会规划、验证、反思，减少对参数量的依赖。
3. **多教师域路由蒸馏缓解模式冲突**：全领域SFT导致搜索（多轮工具）与指令遵循（单轮长思考）冲突，性能下降；OPD通过教师级路由和领域归一化损失有效恢复平衡。
4. **长程任务中多种能力可互相支撑**：如搜索能力帮助科学任务获取外部知识，指令遵循能力帮助约束跟踪。

## 7. 优点

- **创新方向明确**：从“堆参数”转向“堆交互”，提供可复现的低成本路径。
- **基础设施完整**：KAG 将过程级数据（状态、动作、观察、验证）结构化，适用于多种场景的自对弈扩展。
- **蒸馏策略精巧**：显著词汇对齐（SVA）避免全词汇噪声，领域归一化确保跨域平衡。
- **教师训练有针对性**：每个教师采用特定 SFT+RL 策略（如非对称优势设计、动态采样、两阶段训练），最大化领域专长。
- **实验广泛且结果可靠**：覆盖15个基准，与多个1T模型公平对比，并主动讨论复现差异（τ2-Bench）。
