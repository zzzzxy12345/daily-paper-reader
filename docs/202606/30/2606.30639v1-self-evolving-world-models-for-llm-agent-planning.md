## 摘要
世界模型为长周期LLM智能体提供了一种原则性的前瞻能力：在执行前预测行动后果。然而，不可靠的前瞻可能被忽略、误用，甚至降低下游决策质量。本文提出WorldEvolver，一种自我进化的世界模型框架，它在保持下游智能体和所有模型参数冻结的同时，修正其部署时的上下文。WorldEvolver集成三个模块：（i）情节记忆，通过基于检索的模拟利用真实行动转移；（ii）语义记忆，从预测与观察不匹配中提取持久启发式规则；（iii）选择性前瞻，在将低置信度预测纳入智能体推理上下文前进行过滤。我们在ALFWorld和ScienceWorld上评估WorldEvolver，在Word2World上测量世界模型预测准确率，在AgentBoard上测量下游智能体成功率。大量实验表明，WorldEvolver在三个骨干模型上取得了最高预测准确率，并在下游智能体成功率上领先其他世界模型基线，证明测试时记忆修正同时提升了预测保真度和规划性能。

## Abstract
World models offer a principled way to equip long-horizon LLM agents with foresight: predictions of action consequences before execution. However, unreliable foresight can be ignored, misused, or even degrade downstream decision-making. In this paper, we introduce WorldEvolver, a self-evolving world model framework that revises its deployment-time context while keeping the downstream agent and all model parameters frozen. WorldEvolver integrates three modules: (i) Episodic Memory, which exploits real action transitions through retrieval-based simulation; (ii) Semantic Memory, which extracts persistent heuristic rules from prediction-observation mismatches; and (iii) Selective Foresight, which filters low-confidence predictions before integrating them into agent reasoning context. We evaluate WorldEvolver on ALFWorld and ScienceWorld, measuring world model prediction accuracy on Word2World and downstream agent success rate on AgentBoard. Extensive experiments show that WorldEvolver achieves the highest prediction accuracy across three backbones and leads other world model baselines on downstream agent success rate, demonstrating that test-time memory revision enhances both predictive fidelity and planning performance.

## 1. 核心问题与整体含义（研究动机和背景）

- **研究动机**：LLM智能体在长周期任务中常借助世界模型进行前瞻（预测行动后果）以改善规划。然而，部署时环境分布偏移（sim-to-real gap）会导致世界模型的预测不可靠，不可靠的前瞻可能被智能体忽略、误用甚至损害决策（如图2所示，噪声前瞻会降低动作准确率，而完美前瞻才有提升效果）。
- **核心问题**：如何在冻结智能体策略和世界模型参数的条件下，让世界模型在部署时动态适应环境变化，持续提升预测可靠性和下游规划性能。
- **背景意义**：现有方法要么冻结世界模型参数（无法适应偏移），要么通过梯度更新参数（计算成本高、存在灾难性遗忘）。因此需要一种轻量、可审计的在线自适应机制。

## 2. 方法论：核心思想、关键技术细节

- **核心思想**：不更新模型权重，而是通过外部记忆（情节记忆+语义记忆）在推理时动态修正世界模型的上下文，并引入选择性预见机制过滤不可靠预测。
- **关键技术细节**：
  - **情节记忆（Episodic Memory, M_E）**：存储已执行的动作转移三元组`(o_i, a_i, o_{i+1})`。对于候选动作`a_t`，使用动作令牌的Jaccard相似度检索最相似的`k_ME`个过去转移，作为上下文提供给世界模型，使其利用过往实际经验指导预测。
  - **语义记忆（Semantic Memory, M_S）**：当预测与观察不匹配时（因子化状态比较），利用LLM批评器生成启发式规则（如“检查物体不会移动它”），每条规则附带证据分数`e_i`。只有正证据的规则保留在上下文，作为“框架公理”约束未来预测，避免错误改变。
  - **选择性预见（Selective Foresight）**：计算世界模型预测输出的几何平均令牌概率作为置信度`q_t`，仅当`q_t ≥ τ`时将预测结果传递给智能体，否则传递空结果，避免低置信度预测误导决策。
  - **算法流程（Algorithm 1）**：每步先采样草稿动作`a_t^(0)`，检索记忆并预测后果；选择性过滤后，智能体重新采样最终动作`a_t`；若动作改变则重新预测对齐；执行后更新情节记忆，并基于不匹配生成语义规则更新。

## 8. 不足与局限

- **评估范围有限**：仅测试了ALFWorld和ScienceWorld两个文本游戏环境，未涵盖Web导航、代码生成、机器人、多模态等更复杂领域。
