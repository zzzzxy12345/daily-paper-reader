## 摘要
学习准确的价值函数对于强化学习智能体解决长时域、复杂任务具有决定性作用。传统的时序差分学习目标会遭受随时间累积的价值估计偏差，而扩展时域建模方法（如n步TD回溯和Q-chunking）采用固定的时域价值建模方式，往往不够灵活，难以捕捉长时域多阶段任务中的复杂价值结构。本文表明，通过动态时域组合进行价值更新可以产生强大的离线策略学习方案。我们的方法——通过价值拼接实现自适应离线策略学习——用递归的、时域自适应的价值组合取代了固定时域回溯。其关键要素是将价值优化与一个基于未来状态和时域长度的条件辅助价值函数相结合，该辅助函数通过直接数据监督学习，并通过一个拼接策略最优地选择奖励最大化的时域长度和未来子目标，从而实现时域自适应的价值拼接。这种设计能够基于可操作的子目标状态直接估计和组合拼接变长回报，为离线策略优化提供准确且可贪婪利用的价值监督信号。在OGBench的50个任务中，VAST优于固定步长、扩展时域方法以及生成式价值离线强化学习基线，尤其在高度复杂的长时域决策任务中取得了强劲性能。

## Abstract
Learning accurate value functions plays a decisive role for reinforcement learning (RL) agents to solve long-horizon, complex tasks. Conventional temporal-difference (TD) learning objectives suffer from value-estimation bias that accumulates over the horizon, while extended-horizon modeling methods, such as n-step TD backups and Q-chunking, adopt a rigid, fixed-horizon value-modeling recipe that is often not flexible enough to capture complex value structures in long-horizon, multi-stage tasks. In this paper, we show that enabling value updates with dynamic horizon composition can yield a strong offline policy learning scheme. Our method, Horizon Adaptive Offline Policy Learning via VAlue STitching (VAST), replaces fixed-horizon backups with recursive, horizon-adaptive value composition. Its key ingredient is to couple value optimization with a future state- and horizon-length-conditioned auxiliary value function that is learned through direct data supervision, and a stitching policy that optimally selects the reward-maximizing horizon length and future sub-goal to achieve horizon-adaptive value stitching. This design enables direct estimation and compositional "stitching" of variable-length returns grounded in actionable sub-goal states, providing an accurate and greedily exploitable value-supervision signal for offline policy optimization. Across 50 tasks on OGBench, VAST outperforms fixed-step, extended-horizon methods, and generative-value offline RL baselines, achieving strong performance particularly in high-complexity, long-horizon decision-making tasks.

## 1. 核心问题与整体含义（研究动机和背景）

- **问题背景**：在离线强化学习中，传统时序差分（TD）学习在长时域任务中容易产生价值估计偏差的累积，导致学习效率低下。现有的扩展时域方法（如 n-step TD 备份、Q-chunking）虽然通过固定步长加大回溯视野，但存在僵化、不灵活的缺点——它们强制使用固定长度的回报片段，无法根据任务实际需求动态调节时域长度，尤其在多阶段、组合性的复杂任务中容易丢失关键短时域信号或模糊长时域目标。
- **关键动机**：人类决策能够根据情境自适应地选择粗粒度子目标与细粒度即时行动，但现有算法缺少这种“时域自适应”能力。作者希望通过引入一种递归、自适应的价值组合机制，使离线策略学习既保持长时域稳定性，又不牺牲短时域灵活性。
- **核心问题**：如何在离线数据中，让价值函数能够根据不同时域长度和未来子目标状态进行动态“拼接”（stitching），从而为策略优化提供更准确、更可贪心利用的监督信号。

## 2. 论文提出的方法论

### 2.1 核心思想

VAST 的核心是**用递归、时域自适应的价值组合取代固定时域回溯**。它学习一个**时域条件累积回报函数（G-function）**，该函数衡量在给定策略下，从当前状态 s 到目标状态 s⁺ 经过恰好 k 步的期望折扣回报。然后，引入一个**拼接策略（πstitch）**，该策略同时选择最优的拼接时域长度 k 和对应的子目标状态 sₖ，使得以下目标最大化：G(s, sₖ, k) + γᵏ·V(sₖ)。通过这种“拼接”，价值备份不再局限于数据中连续过渡的固定片段，而是可以在不同轨迹之间灵活组合短程高回报段，达到最优价值估计。最后，一个**执行策略（πexec）** 根据拼接策略选定的子目标，通过目标条件模仿学习输出具体动作，并在测试时用拒绝采样进一步优化。

### 2.2 关键技术细节

- **G-function 学习**：定义 G(s, s⁺, k) = E[∑_{t=0}^{k-1} γᵗ r(sₜ, aₜ) | s₀=s, sₖ=s⁺]。通过两个损失联合训练：
  - **蒙特卡洛回归**：直接监督 G 预测与离线轨迹中实际累加回报的差距（式8）。
  - **时域组合性损失**：利用定理1（可分解性）强制 G 满足跨不同中间时域的一致性（式9），增强泛化。
- **拼接贝尔曼算子**（Stitching Bellman Operator，式13）：
  (T^π_stitch V)(s) = max_{k∈{1..K}, sₖ~Γ^π(s,k)} [ G(s,sₖ,k) + γᵏ V(sₖ) ]
  该算子被证明是 γ-收缩映射（定理2），保证收敛到唯一不动点。
- **价值学习**：状态值函数 V 通过期望回归（类似 IQL）拟合上述最大目标（式16-17）。动作值函数 Q 通过标准单步贝尔曼备份（式15）学习。
- **拼接策略 πstitch**：采用**优势加权回归（AWR）** 从数据中提取，并使用**扩散模型**（即噪扩散过程）参数化，处理连续空间中的时域长度 k 和子目标状态 sₖ 的联合预测（式18-19）。
- **执行策略 πexec**：作为**目标条件模仿学习**，直接监督离线数据中当前状态到未来状态的过渡；训练时不涉及价值最大化，仅在测试时通过拒绝采样（式20）从中选出使 Q 值最大的动作，实现两级奖励最大化。

## 6. 主要结论与发现

- **性能优势**：VAST 在 OGBench 50 个任务中**整体表现最佳**，尤其在 **Hard 域（cube-triple、puzzle-4x5 等）**上得分比最强基线（FQL-n/IFQL-n/QC/VALUE-FLOWS）高出2.5倍以上（表1）。在 Basic 域上也取得最佳或持平，特别是 manipulation 域（scene、cube、puzzle）。
- **固定时域方法的缺陷**：通过 toy 导航例子（图2）说明，固定 n-step 或 chunking 可能因时域长度不匹配而丢失关键短时域信号，导致陷入局部最优。VAST 通过自适应拼接克服了这一点。
- **组合性损失的有效性**：消融实验表明，引入组合性损失（λ>0）能显著提升性能，尤其当 λ 在 0.5 附近时达到最佳；过大则有害。
- **两阶段奖励最大化的必要性**：拒绝采样数量 Nrej 从1增加到16以上时性能持续提升，说明执行层对高价值动作的选择是必要的补充。
- **最大时域 K 的影响**：K 取 10-25 通常最佳，过小限制候选集，过大增加 G-function 学习难度。

## 7. 优点

- **方法创新性**：提出时域自适应价值拼接，将辅助值函数（G）与拼接策略（πstitch）有机结合，首次在离线 RL 中实现递归、变时域的价值备份组合。
- **理论贡献**：证明了拼接贝尔曼算子的收缩性（定理2），为收敛性提供了保证；同时严格证明了 G 的时域可分性（定理1）。
- **实验全面性**：在标准且具有挑战性的 OGBench 上进行了大规模多域评估，消融实验覆盖关键组件，结果统计可靠。
- **代码开源**：论文提供了官方 GitHub 仓库，便于复现和后续研究。
- **设计简练**：借鉴 Option 框架思想但避开了复杂结构，通过 G 函数和两个策略模型简洁实现层次式时域自适应；执行策略采用简单行为克隆避免 OOD 问题。
- **实用导向**：测试阶段无需迭代优化，仅需一次前向推理加少量拒绝采样，推理效率可接受。

## 8. 不足与局限

- **计算开销较大**：VAST 同时维护 5 个网络（G、V、Q、πstitch、πexec），且 πstitch 使用扩散模型，训练时间（2.5h/1M步）比轻量方法（如 FQL 0.67h）长；参数也比同类方法多约 30-50%。这对于实时应用或资源受限场景可能构成限制。
- **G-function 学习依赖数据覆盖**：G 函数需要看到足够的未来状态样本来可靠估计。在数据稀疏或缺失某些目标状态的任务中，其泛化能力可能受限（论文未详细探讨数据质量影响）。
- **超参数敏感性**：性能对最大时域 K、组合损失 λ、拒绝采样数 Nrej 等较为敏感；虽然进行了消融，但缺乏自适应的调节策略，实际应用中可能需要针对每个域进行手动调参。
- **未处理随机性奖励与终止条件**：当前设定假设可到达的子目标在数据轨迹中明确存在，对于有随机终止或稀疏/延迟奖励的场景可能表现不佳。

  - 未在基于视觉的高维输入任务上评估。
