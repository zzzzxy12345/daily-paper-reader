## 摘要
LLM智能体在数字环境中遵循实用的执行循环：它们基于结构化状态进行推理、调用工具、检查反馈并修正动作。将这个循环扩展到物理机器人非常困难，因为物理执行是连续的、依赖于具体形态、不确定的，并且受安全限制。现有的具身AI系统在操作、空间理解、导航和人形控制方面取得了进展，但这些能力通常仍然是专门的模块或松散耦合的决策循环。在这项工作中，我们引入了HoloAgent-0，一个用于现实世界机器人部署的统一具身智能体框架。Embodied AgentOS将语言指令转换为可执行的技能图，调度机器人资源，监控执行，并根据运行时反馈触发澄清或重新规划。HoloAgent-0通过三个耦合层组织异构的机器人模型和控制器：用于闭环执行的Embodied AgentOS、用于物理世界基础的3D空间记忆以及用于机器人动作的具身技能。我们在真实硬件上部署了HoloAgent-0，并评估了其空间记忆、长时域导航以及在运动生成、物体搜索、跨机器人协调和移动操作中的闭环执行。

## Abstract
LLM agents follow a practical execution loop in digital environments: they reason over structured states, invoke tools, inspect feedback, and revise actions. Extending this loop to physical robots is difficult because physical execution is continuous, embodiment-dependent, uncertain, and constrained by safety. Existing embodied-AI systems have advanced manipulation, spatial understanding, navigation, and humanoid control, but these capabilities often remain specialized modules or loosely coupled decision loops. In this work, we introduce HoloAgent-0, a unified embodied agent framework for real-world robot deployment. Embodied AgentOS converts language instructions into executable skill graphs, schedules robot resources, monitors execution, and triggers clarification or re-planning from runtime feedback. HoloAgent-0 organizes heterogeneous robot models and controllers through three coupled layers: Embodied AgentOS for closed-loop execution, 3D spatial memory for physical world grounding, and embodied skills for robot action. We deploy HoloAgent-0 on real hardware and evaluate its spatial memory, long-horizon navigation, and closed-loop execution across motion generation, object search, cross-robot coordination, and mobile manipulation.

## 2. 方法论：核心思想、关键技术细节

- **核心思想**：HoloAgent-0 是一个**统一的具身智能体框架**，通过 **三个耦合层** 实现闭环物理执行：
  - **Embodied AgentOS**：运行时层，将语言指令转换为可执行的技能图（skill graph），调度资源，监控执行，根据反馈进行重规划。
  - **Memory Layer**：提供**持久化空间记忆**（几何、拓扑、语义、层次化多模态场景图 HMSG）和**时间记忆**（任务状态、执行轨迹、结果摘要）。
  - **Embodied Skill Layer**：将机器人能力暴露为**类型化、可监控的技能调用**（typed skill calls），包括导航（HoloNavi）、操作（HoloBrain）、全身运动（HoloMotion）、交互、感知等。

- **关键技术细节**：
  - **Embodied AgentOS 设计原则**：闭环优先、记忆中心、类型化技能接口、默认可观测。
  - **ROS2 命令/状态总线**：AgentOS 通过 command/status topics 调度技能，收集反馈。
  - **分布式空间记忆**：
    - 几何记忆：基于 LiDAR（FAST-LIVO）或纯视觉（GeoFlow-SLAM++）构建 metric 点云/网格。
    - 语义记忆：利用 SAM2、SigLIP 在线构建开放词汇 3D 语义地图，并通过**实例关联**（投影 + IoU 匹配）维持对象身份。
    - 层次化场景图（HMSG）：楼层→房间→视角→对象四层结构，支持粗到细的空间检索和 VLM 验证。
  - **导航（HoloNavi）**：结合**层次化对象导航**（LLM 解析 + CLIP 快速匹配）、**在线验证循环**（VLM 慢推理确认目标）、**主动空间探索**（基于前沿点）。形成“快速匹配→VLM验证→失败则探索”的循环。
  - **操作（HoloBrain）**：作为 VLA 后端，接受 pick、place 等技能调用，输出执行状态（未找到物体、抓取失败等）。
  - **全身运动（HoloMotion）**：支持运动追踪和速度追踪，可组合导航与操作技能。
  - **跨机器人协调**：共享记忆和技能调用，AgentOS 可根据能力、位置分配任务。
  - **记忆更新**：新传感器观测或技能结果触发几何、语义、场景图的增量更新；时间记忆追加轨迹。
