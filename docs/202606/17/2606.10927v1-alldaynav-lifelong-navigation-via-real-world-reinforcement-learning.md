## 摘要
终生具身导航在动态环境中要求机器人从碎片化观察中形成持久的场景理解，这对于依赖显式地图或场景图且难以在结构化环境之外泛化的现有方法来说仍然困难。我们提出AllDayNav，一个终生自学习导航框架，通过强化学习将场景动态隐式编码到大型模型的十亿级参数中，并由自进化多模态记忆驱动，该记忆维护和更新视觉关键帧、语义描述和时间上下文，同时自主生成开放词汇指令、图像目标和结构化奖励。在合成和真实环境中的跨房间、跨情节和跨任务场景实验表明，AllDayNav实现了接近100%的成功率，并在路径效率和鲁棒性上持续超越基于地图、VLM和RL的强基线，证明了隐式、记忆驱动的强化学习作为显式地图的可靠终生导航的可扩展替代方案。

## Abstract
Lifelong embodied navigation in dynamic environments requires robots to form persistent scene understanding from fragmentary observations, which remains difficult for existing methods that rely on explicit maps or scene graphs and struggle to generalize beyond structured settings. We propose AllDayNav, a lifelong self-learning navigation framework that implicitly encodes scene dynamics into the billion-scale parameters of a large model via reinforcement learning, powered by a self-evolving multimodal memory that maintains and updates visual keyframes, semantic descriptions, and temporal context while autonomously generating open-vocabulary instructions, image goals, and structured rewards. Experiments in both synthetic and real-world environments across cross-room, cross-episode, and cross-task scenarios show that AllDayNav achieves success rates approaching $100\%$ and consistently surpasses strong map-based, VLM, and RL baselines in path efficiency and robustness, demonstrating implicit, memory-driven reinforcement learning as a scalable alternative to explicit mapping for reliable lifelong navigation.

- **整体含义**：本文探索一种**无需显式建图、无需人类监督**的终生自学习导航范式，让机器人能在完全未知的动态环境中，通过自我交互持续改进导航能力。其意义在于将“隐式记忆+强化学习”作为显式地图的可扩展替代方案，为真实世界中的自主导航提供新思路。

## 二、方法论

### 核心思想
- 受认知科学中**视觉心理意象**启发，将导航目标表示为**图像形式**，构建**图像中心**的持续内部表征。
- 通过**记忆-策略共演化循环**：探索丰富记忆 → 记忆生成自指令和检索目标 → 指令驱动策略学习 → 改进策略促进更高效探索。

### 关键技术细节
1. **自进化多模态记忆数据库**
   - 每条记忆为四元组：(RGB关键帧, 语义描述, 时间戳, LLaVE视觉嵌入)。
   - 关键帧筛选：当前帧嵌入与上一关键帧嵌入余弦相似度小于阈值（0.15）时纳入候选。
   - 去重更新：与记忆库中已有条目比较，若最大相似度 ≤0.7 视为新颖，插入新条目；若 >0.7，则通过图像-文本对齐度判断是否替换视觉部分。
   - VLM语义检索：使用Qwen3-VL-8B-Instruct根据自然语言指令和记忆描述选择最匹配目标，并输出置信度。

2. **预训练VLA骨干网络**
   - **双编码器**：SigLIP（语义）+ DINOv2（结构），特征拼接后经MLP投影至模型维度。
   - **历史帧压缩**：全局平均池化将历史帧变为4个token，配合指数衰减注意力偏置。
   - 输入序列：当前帧（64 token）+ 目标图像（64 token）+ 历史帧（最多64帧×4 token）。
   - **动作表示**：预测8个未来路点（连续坐标和偏转角）。
   - 预训练在Habitat模拟器中收集的50000条轨迹（500万步），80%最短路径专家数据 + 20%随机探索数据，使用MSE损失监督学习。

3. **自指令生成与混合目标检索**
   - 每10情节或记忆增长20%时，调用VLM基于记忆描述生成50条开放词汇导航指令（含对象级、功能级、空间级、时序级等）。
   - 混合检索：优先从记忆库中通过VLM语义检索目标；若置信度<0.6 或探索早期（概率随记忆增长衰减），则回退到互联网图像数据库（G_internet）。

4. **保守Q学习策略优化**
   - **奖励设计**：终端成功奖励（+10，当视觉相似度≥0.75时触发）+ 进度奖励（首次超过阈值0.5/0.6/0.7各+2）+ 时间惩罚(-0.01/步) + 碰撞惩罚(-0.1)。
   - **CQL目标**：标准Bellman误差 + 保守正则项（惩罚OOD动作的Q值），α=5.0。
   - **策略目标**：Q值最大化 + 行为克隆正则化（权重0.1），从成功轨迹中监督。
   - 更新频率：每100步更新一次。

## 三、实验设计
