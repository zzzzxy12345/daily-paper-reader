## 摘要
自动化奖励设计（ARD）旨在通过语言驱动的奖励函数合成来替代强化学习中的人工奖励工程。然而，现有基于大型语言模型（LLM）的方法本质上仍是相关性驱动的，依赖迭代的环境反馈来为每个具体任务优化奖励假设。这种范式不仅导致推理效率低下，还使LLM容易受到语义合理但因果虚假的奖励组件的影响，从而导致优化无效。为解决这些局限，我们提出因果奖励世界模型（CRWM），该模型通过在多任务交互数据上进行离线预训练，显式建模候选奖励组件与任务目标物理变量之间的因果拓扑关系。基于从粗到细的预训练策略，我们引入了一个联合优化模块，该模块将显式机制解耦与置信度感知的软融合相结合，利用微观轨迹细化粗结构先验，从而构建稳健且可解释的因果骨架。在推理阶段，LLM利用CRWM作为任务无关的因果先验来约束奖励生成，实现零样本奖励函数设计。我们的工作为ARD问题开辟了一种新的白盒范式。在复杂连续控制基准上的大量实验表明，CRWM无需基于反馈的奖励细化即可生成可执行的奖励函数，显著降低了获取新机器人技能的设计延迟，同时匹配或超越了最先进的性能，并进一步在未见任务和多种机器人形态上展现出强大的泛化能力。

## Abstract
Automated Reward Design (ARD) aims to replace manual reward engineering in reinforcement learning with language-driven reward function synthesis. However, existing approaches based on large language models (LLMs) remain inherently correlation-driven, relying on iterative environmental feedback to refine reward hypotheses for each specific task. This paradigm not only results in inefficient reasoning but also makes LLMs susceptible to semantically plausible yet causally spurious reward components, leading to ineffective optimization. To address these limitations, we propose the Causal Reward World Model (CRWM), which explicitly models the causal topological relationships between candidate reward components and task-targeted physical variables through offline pre-training on multi-task interaction data. Based on a coarse-to-fine pre-training strategy, we introduce a joint optimization module that integrates Explicit Mechanism Decoupling with Confidence-Aware Soft Fusion to refine coarse structural priors using micro-level trajectories, thereby constructing a robust and interpretable causal skeleton. During inference, LLMs leverage CRWM as a task-irrelevant causal prior to constrain the reward generation, enabling zero-shot reward function design. Our work opens up a new white-box paradigm for the ARD problem. Extensive experiments on complex continuous control benchmarks demonstrate that CRWM generates executable reward functions without feedback-driven reward refinement, significantly reducing the design latency for acquiring new robotic skills while matching or surpassing state-of-the-art performance, and further exhibits strong generalization capabilities across unseen tasks and diverse robotic embodiments.

- **问题背景**：基于大型语言模型（LLM）的自动化奖励设计（ARD）方法虽能减轻人工设计负担，但本质上是**相关性驱动**的，需要迭代的环境反馈（如进化搜索）来筛选奖励假设。这种范式存在两大缺陷：
  - **效率低**：每项新任务都需反复与环境交互（数十次迭代，每次耗时约1小时），延迟高。
  - **易受虚假相关误导**：LLM 会引入语义合理但因果虚假的奖励组件，导致“优化崩溃”（梯度噪声使训练停滞）或“规范博弈”（代理利用虚假信号获得高分却未完成任务）。
- **本文动机**：利用**因果结构**来消除虚假相关。在不同任务中，奖励组件与物理变量之间常共享稳定因果拓扑。若能从多任务数据中提取该拓扑并作为先验，则可使 LLM 在**零样本**条件下直接生成正确奖励，无需在线迭代。
- **整体含义**：提出**因果奖励世界模型（CRWM）**，将奖励设计从“黑箱试错”转变为“白盒因果推理”，为 ARD 开辟新范式。

## 2. 方法论

### 核心思想
- 离线预训练一个任务无关的因果骨架（CRWM），刻画候选奖励组件（如物体位置、速度、接触力）与任务得分之间的因果依赖关系。
- 推理时，LLM 以 CRWM 为显式结构先验，通过**因果剪枝**排除虚假相关，直接生成可执行奖励函数，无需任何环境反馈。

### 关键技术细节（三阶段）

#### 阶段一：结构先验提取（粗粒度）
- **数据收集**：对每个预训练任务，用现有 ARD 框架提取 k 个奖励组件（如目标距离、抓取力），统一归一化为 d 个共享结构变量 V（如 object_position, contact_force）。然后对每个奖励函数采样 N=30 组权重配置（作为干预），训练策略并记录：
  - **宏观数据集 Dmacro**：权重配置 → 最终成功率（fitness score）
  - **微观数据集 Dmicro**：每步各结构变量的真实值（st）
- **因果基础模型**：利用 LimiX 从 Dmacro 中学习粗因果先验 Mprior（邻接矩阵）和置信度矩阵 Mconf（边的可靠程度）。

#### 阶段二：联合优化模块（细粒度）
- 针对粗先验中可能存在的“结构幻觉”（如误加速度项）以及静态拓扑无法捕获状态依赖的瞬态交互（如接触碰撞）问题，提出：
  - **显式机制解耦（EMD）**：
    - 设可学习的不变骨架 Minv（最终保留为 CRWM）。
    - 用 MLP 参数化状态依赖的瞬态调制矩阵 Mtrans(st-1;θ)。
    - 复合瞬时拓扑：Minst(st-1) = Minv ⊙ (1 + Mtrans)。
    - 这样，瞬态交互由 Mtrans 吸收，而 Minv 保持稳定。
  - **置信度感知软融合（Confidence-Aware Soft Fusion）**：
    - 损失函数加入 Lsoft = γ ||Mconf ⊙ (Minv - Mprior)||²_F，对高置信边强约束、低置信边弱约束，允许 Minv 偏离不可靠的粗先验。
  - 整体优化目标：最小化重构误差 + 有向无环正则化（DAGMA）+ 软融合损失，通过增广拉格朗日法求解。
- 收敛后提取 Minv* 作为最终的 CRWM。

#### 阶段三：因果自动化奖励设计（Causal-ARD）
- 将连续矩阵 Minv* 通过 1D 2-means 聚类自动确定阈值 τ，提取活跃边，序列化为三元组 [source, target, effect weight]。
- 将三元组嵌入系统提示，指导 LLM：
  - “只保留有因果路径达到 score 的组件，其它设为0。”
  - 同时根据任务描述调整具体变量名称和权重。
- 输出：一个 Python 函数，可直接用于 RL 训练，无需改写。

### 公式/算法流程（文字说明）
1. 收集多任务干预数据 Dmacro 和 Dmicro。
2. 用 LimiX 从 Dmacro 得粗先验 Mprior, Mconf。
3. 初始化 Minv, θ；循环直到收敛：
   - 从 Dmicro 采样 (st-1, st)。
   - 计算 Minst = Minv ⊙ (1 + Mtrans(st-1))，预测 ŝt = Minst·st-1。
   - 联合损失：MSE + λ·h(Minv) + γ·||Mconf⊙(Minv-Mprior)||²。
   - ALM 更新 Minv, θ。
4. 收敛后提取 Minv*，聚类得阈值，生成因果提示。
5. LLM 零样本生成奖励函数。

- **不足**：
  - 未与 L2R (Language to Rewards)、Text2Reward 等其它 ARD 方法比较。
  - 预训练数据仅基于 14 个同类 Dexterity 任务，覆盖范围有限。
  - 实物实验只验证了奖励对齐，未执行完整 RL 训练（存在 sim-to-real 动态差异风险）。

## 6. 主要结论与发现
1. **零样本可行且高效**：CRWM 引导的 LLM 在 6 个未见 Dexterity 任务上 ESI=0 即达到与迭代方法（Eureka/URDP）相当甚至更高的 SR（如 LiftUnderarm 0.87 vs Eureka 0.78, URDP 0.87）。
2. **因果先验优于语义先验**：ZS-LLM 平均 SR 接近 0，而 CRWM 平均 0.85，差距巨大。用 LLM 语义推断的因果先验（Causal-ARD w. LLM-Prior）仅 0.09，说明需干预数据驱动。
3. **瞬态解耦与软融合关键**：移除 EMD 或 Soft-Fusion 后 SR 下降超 40 个百分点。
4. **可跨形态/环境泛化**：在 ManiSkill2 上零样本 SR 达 0.64-0.89，仅次于 URDP 但无迭代成本。
5. **实际硬件对齐**：CRWM 奖励函数在真实轨迹上 AUC 平均 96.5%，远高于 ZS-LLM 的 53.9%。
6. **减少失败模式**：CRWM 完全消除了优化崩溃和规范博弈（基线方法中分别占 12.6%-21.2% 和 4.7%-10.4%）。

## 7. 优点
- **方法论创新**：首次将因果结构学习与 LLM 奖励设计结合，提出显式机制解耦，兼顾稳定骨架与瞬态交互。
- **白盒可解释**：因果拓扑可可视化，LLM 的剪枝决策有明确依据。
- **零样本高效**：一次离线预训练，后续任务无需逐任务迭代，大幅降低计算延迟。
- **泛化性强**：跨任务、跨实体甚至跨模拟器/实物均有较好表现。
- **实验设计严谨**：消融充分，超参敏感性分析，LLM/因果模型替代验证，实物离线评估。

## 8. 不足与局限
- **预训练成本高**：需大量干预数据（4200 次 RL 训练），对于超大规模任务可能不可持续。
- **组件池覆盖有限**：CRWM 的泛化受限于预定义的结构变量集；若新任务涉及流体、可变形等新物理量，需扩池并重新训练。
- **瞬态解耦的假设**：文中假设瞬态调制独立同分布且零均值，实际物理交互可能存在复杂非平稳性，理论保证有限。
- **缺少双手机器人实物验证**：因缺少硬件，未做真实双手机器人技能训练（仅单臂实物对齐）。
- **与更多基线对比不足**：未包括如 L2R、Text2Reward 等基于 LLM 的奖励设计方法。
- **静态骨架的局限性**：CRWM 是有向无环图，但物理系统可能包含反馈循环，虽用瞬态部分吸收，但严格约束可能遗漏某些因果关系。
