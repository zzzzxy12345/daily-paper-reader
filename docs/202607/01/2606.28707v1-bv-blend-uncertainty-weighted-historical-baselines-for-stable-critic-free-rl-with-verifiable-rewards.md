## 摘要
以组相对策略优化（GRPO）为代表的可验证奖励无评论家强化学习（RLVR）避免了训练价值函数（评论家），相对于基于评论家的PPO流水线，在调整大型语言模型时减少了内存和计算开销。然而，GRPO风格的优势估计依赖于提示局部（同一提示组内）的奖励统计量，可能不稳定。特别是，当提示组中的所有轨迹获得相同奖励时，组内奖励方差为零，组归一化导致该组优势为零，阻碍了使用二元验证器的冷启动情况下的学习。我们提出BV-Blend，一种无评论家框架，通过结合提示局部在策略统计量与语义聚类条件历史矩来稳定优势估计。BV-Blend维护每个聚类的指数移动平均（EMA）跟踪的奖励矩，从均值标准误（SEM）代理导出置信权重，并利用该权重将历史和提示局部基线及方差统计量混合成标准化优势，用于PPO风格的裁剪更新。在可验证推理基准上的实验表明，BV-Blend提高了训练稳定性和性能，并且在组归一化方法可能停滞的情况下保持鲁棒。

## Abstract
Critic-free reinforcement learning with verifiable rewards (RLVR), exemplified by Group Relative Policy Optimization (GRPO), avoids training a value function (critic) and reduces memory and compute overhead relative to critic-based PPO pipelines for aligning large language models. However, GRPO-style advantage estimation depends on prompt-local (within-prompt-group) reward statistics and can be unstable. In particular, when all rollouts in a prompt group receive identical rewards, the within-group reward variance becomes zero, and group normalization yields zero advantages for that group, impeding learning in cold-start regimes with binary verifiers. We introduce BV-Blend, a critic-free framework that stabilizes advantage estimation by combining prompt-local on-policy statistics with semantic-cluster-conditioned historical moments. BV-Blend maintains EMA-tracked reward moments for each cluster, derives a confidence weight from a standard error of the mean (SEM) proxy, and uses this weight to blend historical and prompt-local baseline and variance statistics into a standardized advantage for PPO-style clipped updates. Experiments on verifiable reasoning benchmarks show that BV-Blend improves training stability and performance, and remains robust in regimes where group-normalized methods may stall.

- **核心问题**：在以GRPO为代表的无评论家（critic-free）可验证奖励强化学习（RLVR）中，组归一化的优势估计存在一个严重缺陷：当提示组内的所有轨迹获得相同奖励时（例如在二元验证器冷启动阶段，所有轨迹都失败），组内奖励方差为零，导致标准化后的优势全部归零，从而使该提示组完全失去学习信号，训练停滞。
- **研究背景**：RLVR使用自动验证器（如数学答案匹配、代码单元测试）提供客观奖励，避免了昂贵的人工偏好标注。GRPO通过组内归一化避免了训练价值函数，降低了内存和计算开销，但其优势估计高度依赖瞬时的组内统计量，方差大且不稳定。多个后续工作尝试改进GRPO的稳定性，但仍存在不足。
- **整体含义**：本文旨在解决GRPO在零方差场景下的优势坍缩问题，提出一种无需评论家的优势估计方法，通过引入语义聚类历史信息来稳定学习信号，从而提升RLVR训练的稳定性和最终性能。

## 2. 论文提出的方法论

### 核心思想
- **结合两类信号**：既利用当前提示组的局部在策略（on-policy）统计量，也利用与当前提示语义相似的过去提示组的低方差历史统计量（通过EMA维护）。
- **不确定性加权混合**：通过均值标准误（SEM）代理衡量历史统计量的可靠性，导出置信权重，自适应地混合历史基线和方差与当前局部统计量，然后执行一次标准化得到优势。
- **兼容PPO**：不改变PPO风格的裁剪目标函数，仅修改优势估计器。

### 关键技术细节
1. **提示聚类**：使用冻结的编码器将每个提示映射到固定K-means码本中的簇，码本在RL训练前离线训练并在训练中固定，避免簇漂移。
2. **历史统计量维护**：对每个簇维护EMA均值和EMA二阶原始矩，以及EMA有效计数。首次观测到某个簇时进行冷启动初始化。
3. **不确定性-置信度映射**：定义SEM代理 `SEM_hist(k) = σ_hist(k) / sqrt(N_eff_k + δ_N)`，其中σ_hist(k)为历史标准差，N_eff_k为有效计数。置信权重 `w_k = exp(-SEM_hist(k) / T)`，T控制灵敏度。低不确定性（SEM小）-> w_k接近1（更依赖历史）；高不确定性（SEM大）-> w_k接近0（更依赖局部）。
4. **基线-方差混合与优势计算**：
   - 混合基线：`b(m) = w(m) * μ_hist(k(m)) + (1 - w(m)) * μ_G(m)`
   - 混合方差：`s(m)² = w(m) * v_hist(k(m)) + (1 - w(m)) * σ_G(m)²`
   - 优势：`A_BV,i(m) = (R_i(m) - b(m)) / (s(m) + δ)`
5. **更新顺序**：使用当前批次前的EMA计算置信度；更新政策后再用当前批次更新EMA。

## 6. 论文的主要结论与发现

- **性能提升**：BV-Blend在所有纯策略RLVR方法中取得最佳平均数学推理和OOD泛化性能，并与使用额外数据的混合方法竞争（甚至超越部分方法）。
- **稳定性增强**：BV-Blend在训练过程中保持更高的有效信号比（即非简并标准化比例的提示组比例），奖励曲线更平滑，熵下降更慢，表明探索更持久。
- **核心机制有效**：消融证实了两方面必要：(1) 单纯注入历史信息而不加权反而有害（朴素历史平均性能下降）；(2) 同时使用历史有效计数和历史标准差（即SEM）优于单独使用其中任何一个。
- **鲁棒性**：在三个不同骨干模型上均优于或接近最强基线，在Llama-3.1-8B上表现尤为突出（标准GRPO存在不稳定）。

## 7. 优点（方法或实验设计上的亮点）

- **方法简洁有效**：无需训练评论家，仅需维护每簇EMA和K-means聚类，对现有PPO流水线改动小，易于集成。
- **自适应置信度**：基于SEM的权重自动衡量历史信息的可靠性，避免在历史不可靠时过度信赖，同时能在零方差局部信号下依靠历史保持学习。
- **实验设计严谨**：
  - 使用同一骨干、同一超参数设置进行公平比较（如GRPO与BV-Blend仅优势估计不同）。
  - 除了域内数学推理，还评估OOD泛化，证明方法一般化能力。
  - 训练动态和困难分层分析深入揭示了优势稳定性如何转化为更好的学习行为。
  - 消融实验清晰论证了各设计组件的贡献。

## 8. 不足与局限

- **依赖固定聚类流水线**：性能可能受编码器质量、聚类数K和码本训练语料影响，在跨领域分布漂移时可能退化。
- **额外开销**：需要维护每簇EMA统计量、进行聚类分配和分布式聚合，增加了书签管理和超参数调优（如EMA率γ、温度T、冷启动N0等）。
- **评估领域有限**：主要集中于数学推理，更广泛的领域（如代码、科学问答、多轮工具使用）和不同的验证器设计（如连续奖励）尚未验证。
- **极端稀疏奖励问题**：如果当前提示组和历史簇均缺乏奖励变化（例如所有簇都只给出0或1），BV-Blend仍可能提供有限的学习信号。
